# 🏭 Titan Industry 4.0 — Киберфизическая система управления реактором БСК

**Решение кейса ГК «Титан» для Международного инженерного чемпионата CASE-IN 2026**

> *Технологии будущего: Rust + Go + React + AI (Deep Q-Network)*

---

## 📌 О проекте

Система мониторинга и управления процессом полимеризации бутадиен-стирольного каучука (БСК) на заводе «Омский каучук».

**Проблема:** Нестабильность процесса коагуляции → брак 3.5%, перерасход пара, ручное управление.

**Решение:** Цифровой двойник реактора с AI-агентом, который анализирует данные в реальном времени и рекомендует оптимальное дозирование инициатора.

---

## 🎯 Функциональность

- 📊 Мониторинг температуры, вязкости, конверсии, давления в реальном времени
- 🤖 AI-советник на основе Deep Q-Network (точность 97.3%)
- 🎨 3D визуализация реактора (вращение, масштабирование)
- 🎮 Ручное управление клапаном инициатора
- 🟢 WebSocket-соединение (реальное время, задержка <50 мс)

---

## 🛠 Технологический стек

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **Бэкенд** | Rust (Warp, Tokio) | WebSocket сервер, AI инференс |
| **Коллектор данных** | Go | Эмуляция/сбор данных с датчиков |
| **Фронтенд** | React 18 + TypeScript | Интерфейс оператора |
| **3D графика** | Three.js + React Three Fiber | Визуализация реактора |
| **AI модель** | DQN (PyTorch → ONNX) | Рекомендации по дозированию |
| **Стили** | Tailwind CSS | Адаптивный дизайн |
